# 🛍️ Customer Segmentation using K-Means

## 📌 Project Description
This project is part of my Machine Learning Internship (Task 2).  
The objective is to group customers into different segments based on their purchasing behavior using K-Means Clustering.

## 📊 Dataset
The dataset contains customer details such as:
- Customer ID  
- Gender  
- Age  
- Annual Income (k$)  
- Spending Score (1-100)  

## ⚙️ Steps Performed
1. Loaded dataset using Pandas  
2. Selected important features (Income & Spending Score)  
3. Used Elbow Method to find optimal number of clusters  
4. Applied K-Means clustering  
5. Visualized customer segments  

## 🔧 Technologies Used
- Python  
- Pandas  
- Scikit-learn  
- Matplotlib  

## 📈 Output
- Elbow graph to determine optimal clusters  
- Scatter plot showing customer groups
